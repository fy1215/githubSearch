# GitHub User Search

GitHub REST API を使ったユーザー検索アプリ

## 機能
- ユーザー名検索
- プロフィール表示
- エラーハンドリング
- ローディング表示
- キーボードショートカット（Enter、/）

## 使用技術
- HTML/CSS/JavaScript
- GitHub REST API
- Fetch API
- async/await
